class ckan_api_client.objects.CkanObject[source]¶

Base for other objects manipulated by the API

is_equivalent(other)[source]¶

Used for object comparison.

Will compare all the fields in self and other, and return True if they all match, False otherwise.

Note that this method, differently from the comparison operator, will ignore the id field (we want to make sure the objects are equal, not that they are the same object).
